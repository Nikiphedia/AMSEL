# AMSEL — Tastenkombinationen

## Wiedergabe

| Taste | Aktion |
|-------|--------|
| **Leertaste** | Play / Pause (Haupt-Audio) |
| **Escape** | Stop Wiedergabe |
| **L** | Loop ein/aus |
| **S + Pfeil rechts** | 5 Sekunden vorwaerts springen |
| **S + Pfeil links** | 5 Sekunden rueckwaerts springen |
| **Leertaste gehalten + Klick** | Wiedergabe ab Klickposition starten |

## CapsLock-Modus (Referenz-Audio)

| Taste | Aktion |
|-------|--------|
| **CapsLock** | Umschalten zwischen Haupt-Audio und Referenz-Modus |
| **Leertaste** | Play / Pause der aktuellen Referenz |
| **L** | Referenz-Loop ein/aus |
| **Pfeil links** | Vorherige Referenz auswaehlen |
| **Pfeil rechts** | Naechste Referenz auswaehlen |

*CapsLock-LED am Keyboard zeigt den aktiven Modus. Farbiger Rahmen um das Referenz-Panel als zusaetzliches Feedback.*

## Solo-Modus

| Taste | Aktion |
|-------|--------|
| **Tab** | Naechster Chunk |
| **R gehalten + Tab** | Vorheriger Chunk |

*Solo-Button in der Toolbar aktivieren. Klick auf Chunk zoomt auf volle Breite mit Vor-/Nachlauf (konfigurierbar in Einstellungen).*

## Navigation

| Taste | Aktion |
|-------|--------|
| **Pfeil links** | Viewport nach links scrollen |
| **Pfeil rechts** | Viewport nach rechts scrollen |
| **Ctrl + Pfeil links** | Vorherige Markierung |
| **Ctrl + Pfeil rechts** | Naechste Markierung |
| **+** | Zeitachse reinzoomen |
| **-** | Zeitachse rauszoomen |
| **Ctrl + 0** | Zoom zuruecksetzen |

## Panel-Fokus

| Taste | Aktion |
|-------|--------|
| **1** | Fokus auf Annotations-Panel (nochmal = Fokus aufheben) |
| **2** | Fokus auf Kandidaten-Panel |
| **3** | Fokus auf Audiofiles-Panel |
| **Pfeil hoch** | Vorheriges Element im fokussierten Panel |
| **Pfeil runter** | Naechstes Element im fokussierten Panel |
| **Escape** | Panel-Fokus aufheben |

## Markierungen

| Taste | Aktion |
|-------|--------|
| **Ctrl + A** | Alle Markierungen auswaehlen |
| **Delete / Backspace** | Ausgewaehlte Markierung(en) loeschen |
| **F2** | Label der aktiven Markierung bearbeiten |

## Analyse

| Taste | Aktion |
|-------|--------|
| **F5** | BirdNET Vollanalyse starten |

## Datei / Projekt

| Taste | Aktion |
|-------|--------|
| **Ctrl + S** | Projekt speichern |
| **Ctrl + Z** | Rueckgaengig |
